瓶盖 的 切割 装置 瓶盖 的 切割 装置 ， 包括 底座 ( 10 ) 、 定位 件 ( 20 ) 、 连接 臂 ( 30 ) 、 操作 杆 ( 40 ) 、 上刀件 ( 50 ) 和 上 刀 压簧 ( 60 ) 。 底座 具有 支架 ( 12 ) 和 下 刀部 ( 14 ) 。 定位 件 由 第一 销轴 ( 11 ) 枢接 于 支架 ； 定位 件 具有 定位 板 ( 22 ) 和 连接 板 ( 24 ) ， 定位 板 开设 切割 孔 ( 26 ) 。 连接 臂 的 一端 由 第二 销轴 ( 21 ) 枢接 于 连接 板 ， 另一端 由 第三 销轴 ( 31 ) 枢接 于 操作 杆 的 一端 。 操作 杆上 具有 连接 部 ( 42 ) 。 上刀件 具有 上刀座 ( 52 ) 和 上 刀部 ( 54 ) ， 上刀 座 由 第四 销轴 ( 41 ) 枢接 于 连接 部 。 上刀 压簧 环设 于 上 刀部 ， 其 一端 固定 于 上 刀 座 ， 其 另一端 固定 于 定位 板 。 该 切割 装置 通过 控制 操作 杆 即可 完成 切割 ， 且 切割 过程 中 无需 用手 接触 瓶盖 ， 提高 了 安全性 。
