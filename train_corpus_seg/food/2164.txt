旋转 取料 装置 旋转 取料 装置 ， 包括 一个 料斗 ( 10 ) 和 复数 个 旋转 取料 组件 ( 30 ) 。 料斗 具有 一个 第一 储料 腔 ( 17 ) 和 一个 第二 储料 腔 ( 19 ) ， 料斗 底板 ( 11 ) 具有 复数 个 第一 供料 孔 ( 21 ) 和 一个 第二 供料 孔 ( 22 ) 。 旋转 取料 组件 包括 一个 轴杆 ( 31 ) 、 一个 旋转 取 料件 ( 33 ) 和 一个 接料 嘴 ( 40 ) 。 轴杆 固定 于 料斗 底板 。 旋转 取 料件 旋转 地环套 于轴杆且 贴合 于 料斗 底板 ， 其 具有 两个 沿轴杆 轴向 延伸 的 取料 通孔 ( 35 ) 。 旋转 取 料件 的 周向 边缘 环 设有 边齿 ， 复数 个 旋转 取 料件 串行 连接 且 藉由 边齿 相互 咬合 。 接料 嘴 具有 两个 接料 孔 ( 43 ) ， 取料 通孔 可选择性 地 连通 接料 孔 或 供料 孔 。 上述 装置 在 不 需要 混合 的 情况 下 实现 了 对 不同 固体 颗粒物 料 的 添加 比例 的 精确 控制 。
