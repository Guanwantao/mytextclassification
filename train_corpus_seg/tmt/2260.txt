一种 基于 过压 保护 的 电网 调控 自动 控制系统 本 实用新型 公开 了 一种 基于 过压 保护 的 电网 调控 自动 控制系统 ， 由 主控 站 ( 1 ) 以及 与 主控 站 ( 1 ) 相连接 的 被控 站 ( 2 ) 组成 ； 所述 主控 站 ( 1 ) 由 信息 采集 处理 和 控制 子系统 ( 14 ) ， 分别 与 信息 采集 处理 和 控制 子系统 ( 14 ) 相连接 的 显示器 ( 11 ) 、 控制台 ( 12 ) 、 打印机 ( 13 ) 以及 第一 调制解调器 ( 15 ) 组成 ； 所述 被控 站 ( 2 ) 则 包括 信息 采集 和 命令 执行 子系统 ( 22 ) ， 信息 处理单元 ( 23 ) ， 第二 调制解调器 ( 21 ) ， 调节 单元 ( 25 ) ， 信息 输出 单元 ( 26 ) ， 信息 输入 单元 ( 24 ) 以及 过压 保护 单元 ( 27 ) ； 本 实用新型 可以 对 过电压 进行 处理 ， 使 过电压 被 压制 在 安全 范围 以内 ， 进而 使本 实用新型 可以 更好 的 对 电网 电压 进行 调控 。
