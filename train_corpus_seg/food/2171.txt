冰淇淋 模具 冰淇淋 模具 ， 其 包括 一个 第一 模块 ( 20 ) 和 一个 第二 模块 ( 40 ) 。 第一 模块 ( 20 ) 具有 一个 第一 模型 件 ( 21 ) 和 一个 固接 于 第一 模型 件 ( 21 ) 的 第一 磁吸件 。 第二 模块 ( 40 ) 具有 一个 第二 模型 件 ( 41 ) 和 一个 固接 于 第二 模型 件 ( 41 ) 的 第二 磁吸件 ， 第二 磁吸件 能够 磁性 吸附 第一 磁吸件 。 第一 模型 件 ( 21 ) 和 第二 模型 件 ( 41 ) 能够 藉由 第一 磁吸件 和 第二 磁吸件 的 磁性 吸附 而 拼合 形成 一个 冰淇淋 容腔 ( 11 ) ， 该 冰淇淋 容腔 ( 11 ) 藉由 一个 通道 ( 13 ) 通向 冰淇淋 模具 的 外 表面 。 该 冰淇淋 模具 结构 简单 ， 巧妙 使用 了 第一 磁吸件 和 第二 磁吸件 的 磁性 吸附力 实现 合模 ， 无需 螺栓 固定 ， 操作 简单 快速 ， 且 容易 清洗 。
