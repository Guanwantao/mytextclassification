具有 油量 监测 功能 的 车载 终端 本 实用新型 公开 了 一种 具有 油量 监测 功能 的 车载 终端 ， 包括 处理器 ( 1 ) 、 GPS 单元 ( 2 ) 、 无线通信 单元 ( 3 ) 、 人机交互 单元 ( 4 ) 、 输入输出 接口 ( 5 ) 和 通话 手柄 ( 6 ) ， 所述 GPS 单元 ( 2 ) 、 无线通信 单元 ( 3 ) 、 人机交互 单元 ( 4 ) 、 输入输出 接口 ( 5 ) 和 通话 手柄 ( 6 ) 均 连接 到 处理器 ( 1 ) 上 ， 还 包括 与 处理器 ( 1 ) 相连 的 油位 传感器 ( 7 ) 。 本 实用新型 能够 集 导航 、 汽车 监控 管理 、 油量 监测 为 一体 ， 具有 结构 简单 、 使用方便 等 优点 。
