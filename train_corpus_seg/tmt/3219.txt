一种 喷码 系统 和 喷码 方法 本发明 公开 了 一种 喷码 系统 和 喷码 方法 。 该 喷码 系统 包括 按照 信号 输送顺序 连接 的 外接 电眼 ( 1 ) 、 安装 有 码 生成器 ( 21 ) 的 工控机 ( 2 ) 、 以及 喷码机 ( 3 ) ， 喷码机 ( 3 ) 上 安装 有 自带 电眼 。 该 喷码 方法 包括 当外 接 电眼 ( 1 ) 检测 到 有 包装 要 通过 时 提示 工控机 ( 2 ) ； 工控机 ( 2 ) 上 的 码 生成器 ( 21 ) 收到 外接 电眼 ( 1 ) 的 信号 时 生成 唯一 码 并发 送到 喷码机 ( 3 ) ， 喷码机 ( 3 ) 按 收到 的 顺序 保存 唯一 码 ； 喷码机 ( 3 ) 自带 电眼 检测 到 包装 要 经过 时 ， 将 喷码机 ( 3 ) 中排 在 最 前面 的 唯一 码 喷 印到 包装 上 。 本发明 的 喷码 系统 和 喷码 方法 提高 了 喷码机 的 喷印 速度 和 喷印 准确度 ， 提高 了 最终 对 每个 产品 进行 追溯 的 效果 。
