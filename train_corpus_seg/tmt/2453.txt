分布式 应急 移动 通信 系统核心 交换 设备 本 实用新型 公开 了 一种 分布式 应急 移动 通信 系统核心 交换 设备 ， 所述 分布式 应急 移动 通信 系统核心 交换 设备 包括 ： 主控 单元 板 ( 1 ) ， 与 主控 单元 板 ( 1 ) 相连接 的 协处理器 及 I / O 板 ( 2 ) 、 语音 控制板 ( 3 ) 和 网络 控制板 ( 4 ) ； 本 实用新型 所述 分布式 应急 移动 通信 系统核心 交换 设备 实现 了 传统 蜂窝 移动 通信 系统 中 的 移动 交换 中心 ( MSC ) 、 归属 位置 寄存器 / 拜访 位置 寄存器 ( HLR / VLR ) 、 短消息 中心 ( SMSC ) 、 分组 数据服务 节点 ( PDSN ) 和 鉴权 中心 ( AUC ) 等 的 功能 ， 实现 了 部分 接入网 即 基站 控制器 ( BSC ) 的 功能 ， 实现 了 接入 公众 电信 网络 的 信令 / 媒体 网关 功能 ， 结构紧凑 ， 体积小 、 重量轻 ， 便于 快速 移动 及 应急 通信 使用 ， 使得 在 紧急情况 下 可以 同时 满足 多样 的 应急 通信 需求 。
