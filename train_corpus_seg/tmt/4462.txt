车载 终端 及其 减震 支座 本 实用新型 公开 了 一种 车载 终端 及其 减震 支座 ， 它 包括 车载 终端 ( 1 ) ， 其 特征 在于 ： 它 还 包括 支撑 柱 ( 2 ) ， 安装 板 ( 3 ) 、 终端 背板 ( 4 ) 、 终端 底板 ( 5 ) 、 底座 ( 12 ) 、 多个 减震 支柱 ( 6 ) 和 多个 弹簧 ( 10 ) ， 所述 的 终端 背板 ( 3 ) 为 车载 终端 ( 1 ) 的 外壳 背板 ， 终端 底板 ( 4 ) 为 车载 终端 ( 1 ) 的 外壳 底板 ； 所述 的 支撑 柱 ( 2 ) 固定 在 底座 ( 12 ) 上 ， 所述 的 安装 板 ( 3 ) 固定 在 支撑 柱 ( 2 ) 上部 ， 所述 的 安装版 ( 3 ) 与 终端 背板 ( 4 ) 间 设置 有 减震 支柱 ( 6 ) ， 所述 的 终端 底板 ( 5 ) 与 底座 ( 12 ) 间 设置 有 弹簧 ( 10 ) 。 该 车载 终端 及其 减震 支座 安装简单 ， 能够 进行 上下左右 全方位 防震 ， 能 有效 地 延长 车载 终端 的 使用寿命 。
