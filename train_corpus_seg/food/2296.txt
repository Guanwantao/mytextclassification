冰淇淋 灌注 成型 装置 及 冰淇淋 生产线 冰淇淋 灌注 成型 装置 ， 包括 成型 模具 ( 20 ) 和 灌注 头 ( 10 ) 。 成型 模具 的 一侧 表面 凹陷 形成 有 成型 槽 ( 21 ) ， 成型 槽 由 两个 球面 凹槽 相交 而成 。 灌注 头 包括 外壳 ( 12 ) 、 隔板 ( 13 ) 、 第一 进料 管 ( 14 ) 和 第二 进料 管 ( 15 ) 。 外壳 具有 进料 端 和 一个 出料 端 ， 在 外壳 的 进料 端 具有 第一 进料 口 ( 121 ) 和 第二 进料 口 ( 122 ) 。 在 外壳 ( 12 ) 的 出料 端 具有 出料 口 ( 123 ) ， 出料 口 ( 123 ) 的 形状 对应 于 成型 槽 的 外 轮廓 。 隔板 设置 于 外壳 内 ， 其 位于 第一 进料 口 和 第二 进料 口 之间 。 第一 进料 管 连接 外壳 并 连通 第一 进料 口 。 第二 进料 管 连接 外壳 并 连通 第二 进料 口 。 本 实用新型 还 提供 了 一种 冰淇淋 生产线 ， 能够 快速 高效 地 生产 具有 花生 形状 的 冰淇淋 。
