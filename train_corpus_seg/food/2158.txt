一种 乳制品 含糖量 的 检测 设备 本发明 提供 了 一种 乳制品 含糖量 的 检测 设备 ， 包括 ： 检测 单元 ， 与 检测 单元 ( 2 ) 进行 数据信号 交换 的 数据 处理单元 ( 3 ) ， 显示 单元 ； 还 包括 与 数据 处理单元 ( 3 ) 进行 数据信号 交换 并 被 所述 数据 处理单元 ( 3 ) 控制 的 前 处理单元 ( 1 ) ， 其 进一步 包括 称量 容器 ( 11 ) 、 过滤 装置 ( 12 ) 、 样品 容器 ( 13 ) 、 处理 液 单元 ( 14 ) 和 转化 单元 ( 15 ) ， 过滤 装置 设置 于 称量 容器 和 样品 容器 之间 ， 样品 容器 ( 13 ) 与 转化 单元 ( 15 ) 相连 ， 转化 单元 ( 15 ) 和 称量 容器 ( 11 ) 分别 与 处理 液 单元 ( 14 ) 连接 ； 同时 ， 样品 容器 ( 13 ) 和 转化 单元 ( 15 ) 与 滴定 反应 单元 ( 21 ) 相连 。
