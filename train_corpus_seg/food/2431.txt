海砂 的 清洗 装置 海砂 的 清洗 装置 ， 包括 一个 清洗 槽 ( 20 ) 、 一个 加热 板 ( 24 ) 、 一个排 液阀 ( 26 ) 、 一个 滤网 ( 28 ) 、 一个 搅拌 件 ( 30 ) 、 一个 PH 计 ( 40 ) 、 一个 酸料 罐 ( 52 ) 、 一个 碱料 罐 ( 54 ) 、 一个 清水 罐 ( 56 ) 、 一个 酸料 泵 ( 62 ) 、 一个 碱料 泵 ( 64 ) 及 一个 清水泵 ( 66 ) 。 清洗 槽 形成 一个 清洗 腔 ( 22 ) 。 加热 板够 加热 所述 清洗 腔 内 的 液体 。 排液阀 设置 于 清洗 腔 底部 ， 滤网 设置 于排 液阀 内 。 海砂 的 清洗 装置 ， 有效 解决 了 人工 清洗 海砂 工作 费时费力 、 有害 气雾 影响 工作人员 健康 的 问题 。
