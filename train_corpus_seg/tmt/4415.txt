一种 新型 车载 天线 本 实用新型 公开 了 一种 新型 车载 天线 ， 它 包括 天线 杆 ( 1 ) 、 天线 座 ( 2 ) 、 挡块 ( 4 ) 、 弹簧 ( 5 ) 、 支架 ( 3 ) 以及 底座 ( 7 ) ， 所述 的 天线 杆 ( 1 ) 底部 与 挡 块 ( 4 ) 固定 连接 ， 所述 的 挡 块 ( 4 ) 嵌 在 天线 座 ( 2 ) 的 腔 体内 ， 挡块 ( 4 ) 底部 焊接 有 弹簧 ( 5 ) ， 弹簧 ( 5 ) 与 天线 座 ( 2 ) 底部 连接 ， 天线 座 ( 2 ) 固定 在 支架 ( 3 ) 上 ， 支架 ( 3 ) 固定 在 底座 ( 7 ) 上 。 本 实用新型 天线 在 车上 安装 方便 ， 安装 位置 可调 ； 底部 设置 弹簧 ， 能 在 行车 过程 中 对 天线 防震 ； 采用 伸缩式 拉杆 天线 ， 天线 方向 和 天线 长度 可以 根据 信号 强弱 进行 调节 。
