一种 发酵 装置 一种 发酵 装置 包括 内腔 ( 11 ) 、 外腔 ( 12 ) 、 第一 温度传感器 ( 14 ) 、 调节阀 ( 15 ) 、 控制 设备 ( 10 ) 、 储液 罐 ( 13 ) 以及 加热器 ( 16 ) 。 内腔 具有 进料 口 ( 110 ) 和 出料 口 ( 112 ) ， 外腔 包围 内腔 ， 并 具有 第一 进口 ( 120 ) 和 第一 出口 ( 122 ) ， 调节阀 与 第一 进口 连接 。 储液 罐 具有 第一 出口 的 第二 进口 ( 130 ) 和 一个 与 第一 进口 相连 的 第二 出口 ( 132 ) 。 加热器 是 蒸汽 热交换器 ， 包括 蒸汽 进口 ( 160 ) 、 冷凝水 出口 ( 164 ) 和 蒸汽 调节阀 ( 162 ) 。 控制 设备 根据 第一 温度 感应器 、 第二 温度 感应器 ( 134 ) 控制 调节阀 和 蒸汽 调节阀 ， 保持 内腔 、 外腔 和 储液 罐 的 温度 稳定 ， 确保 内腔 内料 液 在 稳定 温度 环境 下 发酵 ， 达到最佳 发酵 效果 。
