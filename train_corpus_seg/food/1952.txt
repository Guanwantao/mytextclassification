牛奶 发泡 检测器 牛奶 发泡 检测器 ， 它 包括 ： 一个 带有 刻度 的 玻璃 量筒 ( 10 ) 、 一个 固定 玻璃 量筒 ( 10 ) 的 固定架 ( 30 ) 、 一个 可 罩盖 于 玻璃 量筒 ( 10 ) 筒口 的 杯盖 ( 40 ) 和 一个 穿 设于 杯盖 ( 40 ) 的 打泡 柄 ( 20 ) ； 固定架 ( 30 ) 包括 一个 固定 于 玻璃 量筒 ( 10 ) 筒底 的 底座 ( 32 ) 、 一个 环设 于 玻璃 量筒 ( 10 ) 筒口 的 上 环 ( 34 ) 、 和 复数 个 固定 底座 ( 32 ) 和 上 环 ( 34 ) 的 固定 杆 ( 36 ) ； 打泡 柄 ( 20 ) 位于 玻璃 量筒 ( 10 ) 中 的 一端 设有 一个 网状 的 压泡 盘 ( 22 ) ， 该压 泡盘 上 均 布有 复数 个 圆孔 ( 24 ) ， 打泡 柄 ( 20 ) 可 在 玻璃 量筒 ( 10 ) 内 上下 提压 。 牛奶 发泡 检测器 可 兼顾 牛奶 的 发泡 和 持泡 检测 ， 以 减少 操作过程 中 对 测量 精度 的 影响 。
