冰激凌 脆筒 的 成型 装置 及其 成型 方法 冰激凌 脆筒 的 成型 装置 中 ， 托模 ( 20 ) 设有 一个 与 冰激凌 脆筒 的 外形 相适应 的 模腔 ( 22 ) 。 成型 套 ( 30 ) 可 贴合 于 模腔 的 内壁 ， 并 可 从 模腔 中 脱模 。 注料 模组 ( 40 ) 可 将 冰激凌 脆筒 的 原料 加注 至 模腔 中 成型 套 中 。 预压 模组 ( 50 ) 设有 一个 可 进入 成型 套 并 与 原料 接触 的 压头 ( 52 ) ， 且 压头 可 将 成型 套 中 的 原料 压实 。 成型 模组 ( 60 ) 设有 一个 可 进入 成型 套 并 与 压实 的 原料 接触 的 成型 头 ( 62 ) ， 且 成型 头 可 使 成型 套 中 压实 的 原料 形成 一个 用于 盛放 冰激凌 料液 的 容置 腔 。 定型 模组 ( 70 ) 设有 一个 可 进入 容置 腔 的 喷头 ( 72 ) ， 喷头 可向 容置 腔 的 内壁 喷淋 一种 可 使 原料 粘合 的 定型 液 。
