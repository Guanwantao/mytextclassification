吸管 推车 吸管 推车 ， 具有 一个 使用 方向 ， 且 包括 车架 ( 10 ) 、 沥水 平台 ( 20 ) 、 沥 水桶 ( 30 ) 、 集水 盘 ( 40 ) 和 排水管 ( 50 ) 。 车架 具有 承载 部 ( 11 ) 和 支撑 部 ( 12 ) 。 沥水 平台 和 集水 盘 连接 于 承载 部 。 沥水 平台 垂直于 该 使用 方向 ， 且 具有 沥水 通孔 ( 21 ) 。 沥 水桶 设置 于 沥水 平台 背离 支撑 部 的 一侧 ， 且 包括 桶 体 ( 31 ) 、 第一 隔板 ( 33 ) 和 第二 隔板 ( 35 ) 。 桶 体 一端 对应 于 沥水 通孔 。 第一 隔板 和 第二 隔板 均 垂直于 该 使用 方向 。 第一 隔板 具有 第一 卡接 孔 ( 34 ) 。 第二 隔板 具有 第二 卡接 孔 ( 36 ) 。 集水 盘 的 盘面 与 该 使用 方向 的 夹角 大于 90 度 ， 且 具有 连接 排水管 的 排水 孔 ( 41 ) 。 其 可 避免 吸管 晾干 过程 中 反复 转移 ， 可 节省 人力 且 避免 吸管 损坏 。
