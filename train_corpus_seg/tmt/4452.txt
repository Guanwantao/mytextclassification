防 跳跃 限位 车载 终端 支架 本 实用新型 公开 了 一种 防 跳跃 限位 车载 终端 支架 ， 减震 组件 设置 于 上 支承 板 ( 1 ) 与 下 支承 板 ( 2 ) 之间 ， 下 支承 板 ( 2 ) 的 两端 对称 设置 有 导向 柱 ( 3 ) ， 减震 组件 套接 在 导向 柱 ( 3 ) 上 ； 导向 柱 ( 3 ) 通过 安装 块 ( 4 ) 垂直 固定 安装 在 下 支承 板 ( 2 ) 上 ， 上 支承 板 ( 1 ) 上 设置 有 与 导向 柱 ( 3 ) 相配合 的 导向 通孔 ( 5 ) ， 导向 通孔 ( 5 ) 的 内壁 上 套接 有 与 导向 柱 ( 3 ) 相配合 的 导向 套 ( 6 ) ； 所述 导向 柱 ( 3 ) 的 顶部 设置 有防 跳跃 限位 帽 ( 12 ) 。 本 实用新型 导向 柱 顶部 增设 了 限位 帽 ， 可 有效 防止 终端 震动 复位 后 跳跃 掉落 ， 可 有效 保护 车载 终端 ； 导向 柱 可以 起到 良好 的 限位 导向 作用 ， 稳定 的 导向 可以 有效 避免 上下 支承 板 之间 发生 左右 漂移 或 晃动 。
