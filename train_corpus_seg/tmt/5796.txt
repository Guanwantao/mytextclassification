一种 头盔 面罩 本 实用新型 的 头盔 面罩 ， 包括 外壳 ( 1 ) ， 外壳 ( 1 ) 上 设置 有 开孔 ( 5 ) 以及 可 相对 外壳 ( 1 ) 旋转 的 用于 覆盖 开孔 ( 5 ) 的 面罩 ( 2 ) ， 外壳 ( 1 ) 上 固定 有 转轴 ( 3 ) ， 面罩 ( 2 ) 可 相对 旋转 的 安装 于 转轴 ( 3 ) 上 ， 转轴 ( 3 ) 上套 置 有 扭簧 ( 4 ) ， 扭簧 ( 4 ) 的 一端 与 面罩 ( 2 ) 连接 。 本 实用新型 还 包括 与 所述 面罩 ( 2 ) 配合 的 阻尼 块 ( 9 ) ， 所述 阻尼 块 ( 9 ) 用于 与 所述 面罩 ( 2 ) 接触 并 向 所述 面罩 ( 2 ) 施加 摩擦阻力 。 本 实用新型 的 头盔 面罩 的 掀起 采用 包括 扭簧 的 自动 翻转 装置 ， 扭簧 用于 向 面罩 施加 扭转 作用力 ， 以 使 面罩 由 开孔 旋转 移开 ， 通过 扭簧 和 阻尼 的 配合 ， 实现 了 面罩 的 匀速运动 。
