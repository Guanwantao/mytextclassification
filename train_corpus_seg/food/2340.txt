固体 颗粒 的 定量 添加 装置 本 实用新型 提供 了 一种 固体 颗粒 的 定量 添加 装置 ， 包括 容置 固体 颗粒 的 料仓 ( 10 ) 、 出 料器 ( 12 ) 、 分 料器 ( 20 ) 和 行程 调节 件 ( 30 ) 。 料仓 开设 有 出料 口 ( 11 ) 。 出 料器 具有 用于 容置 由 出料 口 排出 的 固体 颗粒 的 容腔 ( 13 ) ， 且 出 料器 还 包括 朝向 容腔 外侧 的 出料 滑道 ( 14 ) 。 分 料器 具有 可 打开 或 关闭 出料 口 的 侧壁 ( 21 ) 和 可 承接 出料 口 排出 的 固体 颗粒 的 分料 斜面 ( 22 ) 。 定量 添加 装置 还 包括 连接 于分 料器 的 行程 调节 件 ( 30 ) 。 行程 调节 件 包括 气缸 ( 32 ) 、 可以 气缸 内 往复 移动 的 柱塞 ( 34 ) 和 与 柱塞 相连 的 连杆 ( 36 ) 。 连杆 与 分 料器 相连接 。 这样 可以 定量 控制 固体 颗粒 的 添加 量 ， 避免 固体 颗粒 破碎 滞留 在 装置 夹缝 内 而 导致 装置 卡死 。
