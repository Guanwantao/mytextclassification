折弯机 后 档料 智能 定位 装置 本 实用新型 公开 了 一种 折弯机 后 档料 智能 定位 装置 ， 包括 运动 平台 ( 1 ) 、 异步电机 ( 2 ) 、 丝杆 ( 3 ) 、 联轴器 ( 4 ) ， 异步电机 ( 2 ) 通过 联轴器 ( 4 ) 连接 丝杆 ( 3 ) ， 运动 平台 ( 1 ) 连接 于丝杆 ( 3 ) 并 通过 丝杆 ( 3 ) 控制 运动 ， 所述 的 智能 定位 装置 还 包括 控制器 ( 5 ) 和 编码器 ( 6 ) ， 所述 的 控制器 ( 5 ) 连接 异步电机 ( 2 ) 和 编码器 ( 6 ) 。 本 实用新型 针对 使用 低成本 的 普通 电机 驱动 方法 ， 通过 使用 控制器 对 普通 电机 的 定位 缺点 进行 补偿 优化 ， 最大 程度 的 使 其 定位 效果 最大化 。 在 控制器 中以 丝杆 的 位置 为 变量 ， 将 整个 丝杆 每个 不同 部位 的 惯性 距离 数据 保存 为 数据库 ， 并且 通过 每次 的 定位 结果 实时 修正 记录 并 保存 惯性 距离 ， 在 每次 的 定位 前 读取 加入 计算 ， 从而 达到 每个 点 都 精确定位 的 目的 。
