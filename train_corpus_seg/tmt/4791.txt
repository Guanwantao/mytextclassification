翻盖式 点读机 本 实用新型 公开 了 一种 翻盖式 点读机 ， 包括 ： 具有 可 转动 连接 的 左 面板 ( 11 ) 和 右 面板 ( 12 ) 的 主机 ( 1 ) 、 播放器 ( 3 ) ； 所述 主机 ( 1 ) 上 设置 有 一 托盘 ( 2 ) ， 所述 播放器 ( 3 ) 安装 于 该 托盘 ( 2 ) 上 ； 所述 托盘 ( 2 ) 底部 与 主机 ( 1 ) 通过 转轴 ( 25 ) 连接 ， 使得 该 托盘 ( 2 ) 可 相对 于 主机 ( 1 ) 在 一定 角度 范围 内向 上 或 向下 翻转 并 任意 停留 于 某 一 角度 ； 在 完全 折叠 状态 下 ， 所述 托盘 ( 2 ) 折叠 于 合并 的 所述 左 面板 ( 11 ) 和 右 面板 ( 12 ) 的 外侧 。 本 实用新型 的 翻盖式 点读机 的 播放器 可 随 托盘 上下 翻转 并 固定 于 某 一 角度 ， 方便 调整 观看 角度 ， 有利于 使用者 保持 正确 的 坐姿 及 保护视力 。
