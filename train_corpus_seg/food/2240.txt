发酵罐 的 无菌 正压 空气 系统 本 实用新型 涉及 一种 发酵罐 的 无菌 正压 空气 系统 ， 其 包括 一个 除菌 过滤器 ( 28 ) 、 一个 吸气 过滤器 ( 22 ) 和 一个 高压 风机 ( 24 ) 。 除菌 过滤器 ( 28 ) 与 发酵罐 ( 10 ) 相连 ， 吸气 过滤器 ( 22 ) 包括 一个 与 外界 空气 相通 的 进气口 和 一个 排出 经 过滤 后 的 空气 的 出气口 ， 高压 风机 ( 24 ) 包括 一个 与 吸气 过滤器 ( 22 ) 的 出气口 相通 的 入口 和 一个 连接 至 除菌 过滤器 ( 28 ) 的 出口 ， 用以 将 吸气 过滤器 ( 22 ) 排出 的 经 过滤 后 的 空气 吹送 至 除菌 过滤器 ( 28 ) 。 其 结构 简单 ， 成本低 ， 可以 延长 除菌 过滤器 的 使用寿命 ， 可 避免 对 发酵罐 造成 二次污染 。
