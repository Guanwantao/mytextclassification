流动 相 试剂瓶 及其 瓶盖 流动 相 试剂瓶 的 瓶盖 ， 包括 外盖 ( 20 ) 、 弹性 垫 ( 30 ) 、 和 内 抵压件 ( 40 ) 。 外盖 具有 盖板 部 ( 21 ) 和 瓶体 连接 部 ( 23 ) 。 瓶体 连接 部 环绕 地 连接 于 盖板 部 并 沿 垂直于 盖板 部 的 方向 延伸 。 盖板 部和瓶体 连接 部 围成 盖腔 ( 25 ) 。 盖板 部上 具有 外 管道 孔 ( 27 ) 。 弹性 垫 贴合 于 盖板 部 朝向 盖腔 的 一侧 ， 其上 具有 对应 于外 管道 孔 的 弹性 孔 ( 31 ) ， 弹性 孔 在 无 外力作用 下 为 闭合 状态 。 内 抵压件 抵压 于 弹性 垫背 离 盖板 部 的 一侧 ， 且 弹性 垫被 抵压 的 部分 环绕 弹性 孔 。 内 抵压件 具有 对应 于 弹性 孔 的 内 管道 孔 ( 45 ) 。 该 瓶盖 能够 与 流动 相管 形成 密封 连接 ， 避免 溶剂 挥发 泄露 。 本 实用新型 还 提供 了 包括 上述 瓶盖 的 流动 相 试剂瓶 。
