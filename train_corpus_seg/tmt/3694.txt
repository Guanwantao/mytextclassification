一种 芯片 管脚 配置 系统 本 实用新型 提供 一种 芯片 管脚 配置 系统 ， 该 系统 包括 ： 共享 配置 单元 和 芯片 ； 其中 ， 芯片 上 设置 一个 或 多个 检测 管脚 ； 共享 配置 单元 设置 一个 或 多个 偏置 端口 ； 一个 或 多个 偏置 端口 分别 与 一个 或 多个 检测 管脚 相连 ； 一个 或 多个 偏置 端口 中 的 任一 一个 偏置 端口 为 一个 或 多个 检测 管脚 提供 其所 需 的 单一 状态 偏置 。 本 实用新型 通过 单个 或 多个 芯片 管脚 的 相同 偏置 需求 只 需 产生 单 一片 外 状态 偏置 ， 通过 共享 配置 的 方法 减小 偏置 的 总 数量 。
