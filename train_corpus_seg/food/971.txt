一种 猪仔 用 防滑 猪蹄 套 为了 解决 猪舍 地面 较为 光滑 ， 易于 造成 猪仔 摔跤 ， 进而 降低 猪仔 成活率 的 问题 ， 本 实用新型 提出 一种 猪仔 用 防滑 猪蹄 套 ， 包括 一个 一端 开口 、 与 猪蹄 相适应 的 袋状 橡胶套 ( 1 ) ； 在 所述 袋状 橡胶套 ( 1 ) 的 封闭 端 设置 有 与 猪蹄 相适应 的 前 支撑 部 ( 2 ) ， 在 前 支撑 部 ( 2 ) 的 侧 上方 设置 有 与 猪蹄 相适应 的 后 支撑 部 ( 3 ) ； 在 所述 前 支撑 部 ( 2 ) 和 后 支撑 部 ( 3 ) 的 外 底面 设置 有 防滑 的 橡胶 楞 ( 5 ) ； 在 袋状 橡胶套 ( 1 ) 的 开口 端 设置 有 弹性 带 ( 4 ) 。
