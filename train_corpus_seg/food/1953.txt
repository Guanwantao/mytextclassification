卷饼 冰淇淋 制品 卷饼 冰淇淋 制品 ， 其 具有 一个 复合 层 ， 该 复合 层 包括 ： 一层 烘焙 包裹 层 ( 12 ) 、 一层 置于 烘焙 包裹 层上 的 夹心 层 ( 14 ) 和 一层 置于 夹心 层上 的 冰淇淋 层 ( 16 ) 。 复合 层 具有 一个 中心 部 ( 103 ) 和 分别 位于 该 中心 部 ( 103 ) 两侧 的 第一 折叠 部 ( 101 ) 和 第二 折叠 部 ( 102 ) ， 第一 折叠 部 ( 101 ) 与 中心 部 ( 103 ) 相对 设置 ， 第二 折叠 部 ( 102 ) 藉由 第一 折叠 部 ( 101 ) 朝向 中心 部 ( 103 ) 。 该卷 饼 冰淇淋 制品 具有 类似 卷饼 的 形态 ， 且 由 多层 不同 原料 构成 ， 口感 丰富 ； 通过 在 外层 设置 烘焙 包裹 层 ， 还 可以 使卷 饼 冰淇淋 制品 在 储存 和 运输 过程 中 保持稳定 的 形态 和 口感 。
