一种 猪舍 用 降温 系统 为了 解决 现有 猪舍 降温 系统 ， 在 降温 的 同时 使 空气 湿度 明显 增大 的 问题 ， 本 实用新型 提出 一种 猪舍 用 降温 系统 ， 在 猪舍 相对 的 墙壁 上 分别 设置 有 降温 装置 ( 4 ) 和 通风 窗 ( 2 ) ； 所述 降温 装置 ( 4 ) 包括 风扇 ( 5 ) 、 制冷 装置 ( 6 ) 、 集水 装置 ( 7 ) ； 所述 制冷 装置 ( 6 ) 包括 带孔 的 金属 框及 依次 设置 在 金属 框 内部 的 水管 ( 1 ) 、 制冷 材料 ( 8 ) 、 第一 水槽 ( 11 ) ； 在 所述 制冷 材料 ( 8 ) 的 上方 设置 有 可 持续 滴水 的 水管 ( 1 ) ； 所述 制冷 材料 ( 8 ) 为 透气 的 可 持续 吸水 的 网状 物质 ； 所述 集水 装置 ( 7 ) 包括 带孔 的 金属 框及 依次 设置 在 金属 框 内部 的 集水 材料 ( 9 ) 和 第二 水槽 ( 12 ) ； 所述 集水 材料 ( 9 ) 为 透气 的 可 快速 吸水 的 网状 物质 。
