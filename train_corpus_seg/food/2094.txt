试剂瓶 及其 瓶盖 试剂瓶 的 瓶盖 ， 包括 第一 盖板 ( 10 ) 、 第二 盖板 ( 20 ) 、 弹性 件 ( 30 ) 和 瓶体 连接件 ( 40 ) 。 第一 盖板 有 凹腔 ( 11 ) 、 第一 接触面 ( 13 ) 、 第一 通孔 ( 15 ) 和 塞腔 ( 17 ) 。 第二 盖板 有 凸块 ( 21 ) 、 第二 接触面 ( 23 ) 、 第二 通孔 ( 25 ) 和 孔塞 ( 27 ) 。 凸块 与 凹腔 同轴 且 过渡 配合 ， 第一 盖板 和 第二 盖板 可 相对 转动 ， 且 具有 一个 第一 相对 位置 和 一个 第二 相对 位置 。 凸块 ( 21 ) 具有 柱心腔 ( 22 ) ， 弹性 件 设置 于柱 心腔 并 连接 第一 盖板 和 第二 盖板 。 第二 接触面 可 与 第一 接触面 面 接触 。 处于 第一 相对 位置 时 ， 第二 通孔 与 第一 通孔 连通 ， 孔塞 位于 塞腔 内 。 处于 第二 相对 位置 时 ， 孔塞 封闭 第一 通孔 。 该 瓶盖 可 防止 试剂 挥发 。 此外 还 提供 了 包括 该 瓶盖 的 试剂瓶 。
