饲料 加工 装置 本 实用新型 实施 例 公开 了 一种 饲料 加工 装置 ， 包括 饲料 混合机 ( 1 ) 和 饲料 粉碎机 ( 3 ) ， 所述 饲料 混合机 ( 1 ) 连接 所述 饲料 粉碎机 ( 3 ) ， 其中 ， 所述 饲料 粉碎机 ( 3 ) 的 粉碎机 进料 口 ( 4 ) 连接 所述 饲料 混合机 ( 1 ) 的 混合机 出料 口 ( 2 ) ， 所述 饲料 粉碎机 ( 3 ) 连接 有 除尘 布袋 ； 所述 饲料 粉碎机 ( 3 ) 的 粉碎机 出料 口 设置 有 横向 输送机 ( 7 ) 。 通过 本 实用新型 实施 例 提供 的 饲料 加工 装置 ， 由该 饲料 混合机 和 饲料 粉碎机 分别 对 饲料 进行 搅拌 、 混合 、 粉碎 和 输送 ， 其中 通过 饲料 粉碎机 再次 对 从 饲料 混合机 进入 饲料 粉碎机 的 饲料 进行 粉碎 ， 从而 消除 饲料 中 含有 油 坨 的 饲料 ， 使 饲料 与 油脂 的 混合 更加 均匀 ， 有效 提高 浓缩 饲料 的 质量 。
