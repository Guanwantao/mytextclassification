可 调节 式 回吸 装置 本 实用新型 提供 了 一种 可 调节 式 回吸 装置 ， 包括 一个 阀腔 ( 16 ) 、 一个 阀 芯 ( 32 ) 、 一个 回吸 限位 套 ( 22 ) 和 至少 一个 回吸 调整 套 ( 24 ) 。 阀腔 的 一端 与 喷涂 口 ( 17 ) 相连 ， 阀腔 的 周壁 与 料管 ( 18 ) 相连 。 阀 芯 可在阀 腔 内 往复运动 ， 阀 芯 的 一端 具有 沿 轴向 设置 的 一个 阀杆 ( 34 ) ， 回吸 限位 套 和 回吸 调整 套 设置 在阀 杆 上 。 阀 芯 的 外周 壁与阀 腔 的 内 周壁 之间 具有 一个 容腔 ( 36 ) ， 喷涂 口 可 经由 容腔 与 料管 相通 ， 而 当阀 芯 的 外周 壁 触及 阀腔 的 内 周壁 时 ， 容腔 仅 与 料管 连通 。 阀 芯 的 轴向 端面 触及 回吸 调整 阀 的 轴向 端面 时 ， 阀 芯 达到 上止 点 。 通过 改变 回吸 调整 套 的 轴 长度 或 使用 的 数量 ， 可以 调节 回吸量 。
