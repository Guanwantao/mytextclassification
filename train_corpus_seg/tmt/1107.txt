一种 电子 听诊器 本发明 提出 一种 电子 听诊器 ， 包括 上 壳体 ( 1 ) 、 下 壳体 ( 2 ) ， 听诊器 膜 ( 3 ) 固定 在 下 壳体 ( 2 ) 上 用于 接触 人体 而 传导 心音 或 肺音 ， 腔体 罩 ( 6 ) 与 下 壳体 ( 2 ) 紧密连接 并 与 听诊器 膜 ( 3 ) 合围 成聚 声腔 ， 传声器 ( 4 ) 置于 聚 声腔 中 并 固定 在 腔体 罩 ( 6 ) 上 ， 腔体 罩 ( 6 ) 上 具有 通孔 ( 62 ) 使聚 声腔 内部 与 外部 连通 ， 该 听诊器 进一步 包括 隔离 罩 ( 7 ) ， 隔离 罩 ( 7 ) 罩 设在 腔体 罩 ( 6 ) 外部 ， 留有 使 通孔 ( 62 ) 与 外界 导通 的 通道 ， 上 壳体 1 与 下 壳体 2 固接 为 一体 ， 气流 通过 上 壳体 ( 1 ) 上 的 泄 压孔 ( 12 ) 与 外界 导通 ； 通过 设置 泄压 通道 及 隔离 罩 等 结构 ， 可 平衡 听诊器 的 内外 气压 压强 并 隔绝 噪声 ， 以 提升 听诊器 的 灵敏度 及 信噪比 。
