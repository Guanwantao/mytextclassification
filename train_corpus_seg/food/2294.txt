恒温 振荡 水浴锅 本 实用新型 涉及 一种 恒温 振荡 水浴锅 ， 其 包括 一个 摇床 ( 10 ) 、 一个 承载 架 ( 20 ) 和 一个 可 连接 摇床 和 承载 架 的 条状 连接件 ( 30 ) 。 摇床 的 两个 相对 的 侧壁 上 分别 设置 有 一个 第一 固定 部 ( 12 ) 和 一个 第二 固定 部 ( 14 ) 。 承载 架上 设置 有 两个 第一 连接 孔 ( 22 ) 和 两个 第二 连接 孔 ( 24 ) 。 条状 连接件 包括 一个 第一 连接端 ( 32 ) 和 一个 第二 连接端 ( 34 ) ， 第一 连接端 具有 两个 第一 连接 部 ( 322 ) ， 各 第一 连接 部可 分别 穿 设过 一个 第一 连接 孔以 与 第一 固定 部 连接 ， 第二 连接端 具有 两个 第二 连接 部 ( 342 ) ， 各 第二 连接 部可 分别 穿 设过 一个 第二 连接 孔以 与 第二 固定 部 连接 。 恒温 振荡 水浴锅 利用 条状 连接件 连接 承载 架 和 摇床 的 方式 简单 ， 容易 固定 ， 制造 成本低 。
