货架 跟踪 ( OST ) 系统 安装 在 销售 单元 上 的 系统 具有 一个 或 多个 库存 区域 、 一个 或 多个 产品 单元 、 一个 或 多个 产品 传感器 、 安装 结构 和 电磁 信号处理器 。 在 多个 不同 的 布置 中 ， 一个 或 多个 库存 区域 能够 呈现 一个 或 多个 产品 单元 。 每个 至少 与 一个 或 多个 库存 区域 中 的 一个 可 操作 地 关联 的 一个 或 多个 产品 传感器 将 一个 或 多个 产品 单元 的 感 测量 转换成 相应 的 模拟 电磁 信号 。 安装 结构 将 一个 或 多个 产品 传感器 相对 于 一个 或 多个 产品 单元 紧固 到 销售 单元 上 ， 以 使 一个 或 多个 产品 传感器 感测 一个 或 多个 产品 单元 的 量 。 与 一个 或 多个 产品 传感器 通信 的 电磁 信号处理器 能够 对 来自 一个 或 多个 产品 传感器 的 输出 进行 周期性地 采样 ， 并且 将 模拟信号 转换成 数字信号 。
