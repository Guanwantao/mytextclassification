用于 耳机 的 线控 装置 及 耳机 一种 用于 耳机 的 线控 装置 包括 滤波 单元 ( 200 ) 、 控制 单元 ( 100 ) 、 第一 电阻 ( 300 ) 、 第二 电阻 ( 400 ) 、 第一 开关 ( 500 ) 、 第二 开关 ( 600 ) 及 第一 电容 ( 700 ) 。 第一 电阻 ( 300 ) 与 第一 开关 ( 500 ) 依次 串联 于 电源 与 接地 之间 ， 第一 电阻 ( 300 ) 与 第一 开关 500 的 公共 连接点 接 控制 单元 ( 100 ) 的 第一 端 ( 101 ) 。 第二 电阻 ( 400 ) 与 第二 开关 ( 600 ) 依次 串联 于 电源 与 接地 之间 ， 第二 电阻 ( 400 ) 与 第二 开关 ( 600 ) 的 公共 连接点 接 控制 单元 ( 100 ) 的 第二 端 ( 102 ) 。 当 第一 开关 ( 500 ) 或 第二 开关 ( 600 ) 由 闭合 到 断开 时 ， 控制 单元 ( 100 ) 输出 预设 编码 脉冲 信号 ； 当 第一 开关 ( 500 ) 或 第二 开关 ( 600 ) 闭合 时 ， 与 线控 装置 连接 的 外接 终端 根据 第一 电阻 ( 300 ) 或 第二 电阻 ( 400 ) 上 的 压降 执行 预设 操作 。
