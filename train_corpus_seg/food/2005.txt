提手 静态 拉力 检测 装置 本 实用新型 提供 了 提手 静态 拉力 检测 装置 ， 包括 ： 一个 支撑架 ( 10 ) 、 一个 连接 支撑架 ( 10 ) 的 挂 杆 ( 20 ) 和 一个 计时 机构 ( 30 ) 。 计时 机构 ( 30 ) 包括 ： 一个 光电 传感器 ( 32 ) 、 一个 处理器 ( 33 ) 和 一个 显示器 ( 34 ) 。 光电 传感器 ( 32 ) 位于 挂杆 ( 20 ) 的 一侧 。 处理器 ( 33 ) 具有 计时 功能 ， 其 与 光电 传感器 ( 32 ) 可 信号 连接 。 显示器 ( 34 ) 与 处理器 ( 33 ) 可 信号 连接 。 该 提手 静态 拉力 检测 装置 使用 方便快捷 ， 且 检测 精度 较 高 。
