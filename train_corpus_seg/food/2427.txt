勺子 本 实用新型 提供 了 一种 勺子 ， 勺子 包括 ： 勺头 ( 10 ) ； 勺 柄 ( 20 ) ， 包括 固定 部 ( 21 ) 和 伸缩 部 ( 22 ) ， 固定 部 ( 21 ) 与 勺 头 ( 10 ) 连接 ， 伸缩 部 ( 22 ) 穿 设在 固定 部 ( 21 ) 中 ， 勺 柄 ( 20 ) 具有 最短 位置 和 最长 位置 ， 勺 柄 ( 20 ) 位于 最短 位置 时 伸缩 部 ( 22 ) 的 靠近 勺头 ( 10 ) 的 一端 可 延伸 至勺头 ( 10 ) 所在 的 一侧 。 本 实用新型 的 技术 方案 有效 的 解决 了 现有 技术 中 勺子 的 长度 满足 正常 使用 时 勺子 过长 不能 放在 杯盖 中 的 问题 。
