分装 装置 分装 装置 ， 包括 水平 传送带 ( 11 ) 、 第一 止挡 组件 ( 12 ) 、 第二 止挡 组件 ( 15 ) 、 料罐 ( 20 ) 、 分装 头 ( 21 ) 、 第一 感应 件 ( 31 ) 、 第二 感应 件 ( 32 ) 、 和 控制 单元 ( 33 ) 。 水平 传送带 用于 传送 容器 ， 其 具有 进入 工位 、 灌装 工位 和 传送 方向 。 第一 止挡 组件 设置 于 进入 工位 ， 其 包括 第一 挡板 ( 13 ) 和 第一 升降 驱动 件 ( 14 ) 。 第二 止挡 组件 设置 于 灌装 工位 ， 其 包括 第二 挡板 ( 16 ) 和 第二 升降 驱动 件 ( 17 ) 。 分装 头 连通 料罐 。 第一 感应 件 设置 于 进入 工位 。 第二 感应 件 设置 于 灌装 工位 。 控制 单元 信号 连接 第一 感应 件 、 第二 感应 件 、 第一 升降 驱动 件 、 第二 升降 驱动 件 、 和 分装 头 。 该 分装 装置 可 将 待 分装 物 自动 且 精确 地 分装 至 多个 容器 。
