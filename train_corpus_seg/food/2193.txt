往复式 撒 颗粒 装置 一种 往复式 撒 颗粒 装置 ， 包括 颗粒 料斗 ( 10 ) 、 液态 料槽 ( 20 ) 、 伸缩 管 ( 30 ) 和 往复 移动 件 ( 40 ) 。 伸缩 管 包括 柔性 管部 ( 31 ) 和 刚性 管部 ( 33 ) 。 柔性 管部 具有 连接 颗粒 料斗 的 料斗 连接端 ( 312 ) 和 插接 端 ( 314 ) 。 刚性 管部 具有 套 设于 插接 端的 套接 端 ( 332 ) 和 出料 端 ( 334 ) 。 往复 移动 件 ( 40 ) 包括 位于 液态 料槽 上方 的 固定 杆 ( 42 ) 和 在 固定 杆上 往复 移动 的 移动 块 ( 44 ) ， 移动 块 可 夹持 出料 端 随之 运动 ， 以 将 颗粒 撒向 液态 料槽 。 本 实用新型 提供 的 往复式 撒 颗粒 装置 ， 利于 颗粒 与 冰淇淋 料液 的 快速 混合 ， 缩短 颗粒 与 冰淇淋 料液 的 混合 时间 ， 避免 颗粒 从料 液 中间 吸收 水分 ， 保证 颗粒 的 脆 硬度 。
