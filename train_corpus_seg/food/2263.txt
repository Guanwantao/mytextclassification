温度计 比 对 悬挂 架 温度计 比 对 悬挂 架 ， 其 具有 一个 使用 方向 ， 且 包括 一个 基座 ( 10 ) 和 一个 升降 架 ( 20 ) 。 基座 ( 10 ) 具有 一个 限位 板 ( 11 ) ， 限位 板 ( 11 ) 在 该 使用 方向 上 穿 设有 复数 个 限位 孔 ( 12 ) 。 升降 架 ( 20 ) 包括 一个 固定 件 、 一个 升降 件 ( 23 ) 、 和 一个 紧锁 件 。 固定 件 固接 于 基座 ( 10 ) 。 升降 件 ( 23 ) 可 沿该 使用 方向 滑动 地 连接 于 固定 件 。 升降 件 ( 23 ) 上 设有 复数 个 在 该 使用 方向 上 对应 于 限位 孔 ( 12 ) 的 挂钩 ( 25 ) 。 紧锁 件 连接 于 固定 件 ， 其 能够 将 升降 件 ( 23 ) 抵压 固定 于 固定 件 。 该 温度计 比 对 悬挂 架 ， 可 将 温度计 悬挂 于 恒温 水浴 ， 且 适用 于 不同 长度 的 温度计 ， 具有 较 好 的 悬挂 稳定性 。
