一种 大规模 车辆 GPS 数据 订阅 发布 的 方法 本发明 公开 一种 大规模 车辆 GPS 数据 订阅 发布 的 方法 , 包括 步骤 ： ( 1 ) 解析 车机 上报 的 GPS 数据 信息 ， 获得 车机 MAC 、 地域 ID ； ( 2 ) 根据 车机 MAC 查询 HASH 表 获得 订阅 者 列表 ； ( 3 ) 根据 地域 ID 获得 订阅 者 列表 ； ( 4 ) 判断 车机 的 地域 ID 为 市域 ID 还是 省域 ID ： 为 省域 ID 执行 步骤 ( 5 ) ； 为 市域 ID 执行 步骤 ( 6 ) ； ( 5 ) 根据 省域 ID 查询 HASH 获得 订阅 者 列表 ， 转 步骤 ( 8 ) ； ( 6 ) 根据 市域 ID 查询 HASH 获得 订阅 者 列表 ； ( 7 ) 根据 市域 ID 匹配 获得 省域 ID ， 再 根据 省域 ID 查询 HASH 获得 订阅 者 列表 ； ( 8 ) 去除 上述 根据 车机 MAC 、 地域 ID 获得 的 订阅 者 列表 中 的 重复 数据 ； ( 9 ) 遍历 订阅 者 列表 ， 分发 车辆 GPS 数据 。
