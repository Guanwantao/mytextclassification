一种 终端 信号 检测 方法 、 电路 及 装置 本发明 涉及 一种 终端 信号 检测 方法 、 电路 及 装置 ， 本发明 的 一种 装置 实施 例 包括 USB 驱动 ( 11 ) 、 终端设备 ( 12 ) 和 终端 信号 检测 电路 ( 13 ) ； 终端 信号 检测 电路 ( 13 ) 对 USB 驱动 ( 11 ) 与 终端设备 ( 12 ) 的 连接 状态 进行 判断 ， 获取 USB 驱动 ( 11 ) 与 终端设备 ( 12 ) 连接 节点 的 节点 电压 ( V1 ) ， 终端 信号 检测 电路 ( 13 ) 根据 节点 电压 ( V1 ) 获得 第三 输出 信号 ( OUT3 ) 并 输出 至 USB 驱动 ( 11 ) ， USB 驱动 ( 11 ) 受控 于 第三 输出 信号 ( OUT3 ) 。 本发明 提供 一种 可靠 的 终端 信号 检测 方法 、 电路 及 装置 ， 以 保证 在 不同 的 终端 接口 连接 方式 下 ， 状态机 能够 正确 切换 USB 驱动 的 工作 状态 。
