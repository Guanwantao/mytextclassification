一种 防水 指纹 门禁 装置 本 实用新型 所 提供 一种 具有 多重 防水 功能 的 指纹 门禁 装置 ， 并 具有 针对性 的 对 设备 多个 关键部位 进行 防水 处理 ， 以 提高 产品 整体 的 防水 效果 及 等级 。 应用 本 技术 所 设计 的 产品 具有 价格合理 、 结构 简单 ， 防水 、 防尘 等级 高 、 方便 安装 及 维修 等 优势 ， 并 适合 产品 的 大规模 加工 及 生产 。 本 实用新型 的 是 这样 实现 的 一种 防水 指纹 控制器 ， 主要 包括 刷卡 设备 ( 1 ) 、 指纹 设备 ( 2 ) 、 声音 设备 ( 3 ) 、 第一 数据 接口 ( 4 ) 、 第二 数据 接口 ( 5 ) 、 开关 ( 6 ) 、 前壳 ( 10 ) 、 后壳 ( 20 ) 、 挂板 ( 7 ) 等 组成 ， 其 特征 在于 ： 所述 前壳 ( 10 ) 内 设置 有 刷卡 设备 ( 1 ) 、 指纹 设备 ( 2 ) 、 声音 设备 ( 3 ) 、 第一 数据 接口 ( 4 ) 等 设备 ； 所述 后壳 内 设置 有 第二 数据 接口 ( 5 ) 、 开关 ( 6 ) 等 设备 ； 所述 以上 设备 安装 时 设置 有 1 个 或 多个 防水 装置 ； 所述 挂板 ( 7 ) 被扣 合在 后 壳 ( 20 ) 的 外部 。
