一种 车用 空气 净化器 控制系统 本 实用新型 公开 了 一种 车用 空气 净化器 控制系统 ， 包括 ： 汽车 电源 ( 1 ) 、 电源 转化 模块 ( 2 ) 、 气体 传感器 模块 ( 3 ) 、 温湿度 检测 模块 ( 4 ) 、 键盘输入 模块 ( 5 ) 、 微处理器 ( 6 ) 、 负离子 发生器 模块 ( 7 ) 、 紫外光 模块 ( 8 ) 、 通风 装置 ( 9 ) 、 液晶显示 模块 ( 10 ) 和 数据 存储模块 ( 11 ) ； 气体 传感器 模块 ( 3 ) 、 温湿度 检测 模块 ( 4 ) 实时 采集 车 内 空气质量 指标 、 温湿度 信息 输出 至 微处理器 ( 6 ) ， 微处理器 ( 6 ) 将 所述 指标 信息 分析 处理 后 得到 数据 结果 输出 至 液晶显示 模块 ( 10 ) 和 数据 存储模块 ( 11 ) ， 并 根据 所述 数据 结果 ， 来 控制 负离子 发生器 模块 ( 7 ) 、 紫外光 模块 ( 8 ) 、 通风 装置 ( 9 ) 的 工作 状态 。 本 实用新型 能 对 车内 空气质量 实时 监控 ， 测量 精度高 ， 具有 杀菌 除味 、 自动 通风 功能 ， 有益于 驾驶者 精神 集中 、 心情舒畅 ， 更 有益于 驾乘 人员 的 身体健康 。
