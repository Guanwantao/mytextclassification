带 双处理器 的 车载 终端 本发明 公开 了 一种 带 双处理器 的 车载 终端 ， 包括 主 处理器 ( 1 ) 、 GPS 单元 ( 2 ) 、 无线通信 单元 ( 3 ) 、 人机交互 单元 ( 4 ) 、 输入输出 接口 ( 5 ) 、 防盗器 ( 7 ) 和 供电 单元 ( 8 ) ， 所述 GPS 单元 ( 2 ) 、 无线通信 单元 ( 3 ) 、 人机交互 单元 ( 4 ) 、 输入输出 接口 ( 5 ) 和 防盗器 ( 7 ) 均 连接 到 主 处理器 ( 1 ) 和 供电 单元 ( 8 ) 上 ， 所述 供电 单元 ( 8 ) 还 与 主 处理器 ( 1 ) 相连 ， 还 包括 从 处理器 ( 6 ) ， 所述 从 处理器 ( 6 ) 与 主 处理器 ( 1 ) 和 供电 单元 ( 8 ) 均 相连 。 本发明 的 车载 终端 增设 了 一个 从 处理器 管理系统 电源 ， 并 在 检测 到 系统 异常 时 或者 定期 在 系统 空闲 时 断电 重启 ， 保证系统 的 稳定性 ， 具有 结构 简单 、 使用方便 等 优点 。
