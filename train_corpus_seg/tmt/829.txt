带 外罩 的 车用 摄像头 本 实用新型 公开 了 一种 带有 外罩 的 车用 摄像头 ， 包括 壳体 ( 1 ) 、 摄像头 组件 ( 3 ) 、 外罩 ( 5 ) 和 位于 壳体 ( 1 ) 两侧 的 底座 ( 4 ) ， 所述 的 外罩 ( 5 ) 固定 在 壳体 ( 1 ) 的 外部 ， 所述 的 壳体 ( 1 ) 设置 有 内 凹 的 与 摄像头 组件 ( 3 ) 对应 大小 的 腔体 ， 摄像头 组件 ( 3 ) 位于 壳体 ( 1 ) 的 腔体 内部 ， 摄像头 组件 ( 3 ) 可以 在 腔 体内 转动 。 本 实用新型 通过 上述 结构 ， 摄像头 外部 设置 有 保护 罩 ， 使用 更 安全 。
