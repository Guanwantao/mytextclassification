一种 分布式 麦克风 智能家居 系统 本 实用新型 提出 一种 分布式 麦克风 智能家居 系统 ， 包括 分布 安装 在 家庭 或 单位 内 的 一个 或 多个 房间内 的 麦克风 装置 ( 1 ) 及 安装 在 其中 一个 房间内 的 一台 中心 处理器 ( 2 ) ， 其 特征 在于 ， 每个 房间内 布置 一个 或 多个 麦克风 装置 ( 1 ) ， 其 用于 采集 房间内 说话 人 的 语音 并 将 该 语音 信号 通过 连接 链路 发送至 中心 处理器 ( 2 ) ， 中心 处理器 ( 2 ) 对 说话 人 的 声音 信号 进行 处理 ， 根据 说话 人 的 语音指令 来 控制 与 中心 处理器 ( 2 ) 连接 的 家居 产品 ； 所述 一个 或 多个 麦克风 装置 ( 1 ) 接入 家庭 的 电力 线路 中 ， 该 一个 或 多个 麦克风 装置 ( 1 ) 与 中心 处理器 ( 2 ) 通过 电力线 ( 3 ) 相连接 ， 麦克风 装置 ( 1 ) 采集 的 信号 经由 电力线 ( 3 ) 发送至 中心 处理器 ( 2 ) ； 使用 本 实用新型 的 分布式 麦克风 智能家居 系统 ， 可 实现 便捷 的 智能家居 控制 。
