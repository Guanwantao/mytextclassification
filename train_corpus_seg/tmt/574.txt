微波 信号 复用 电路 一种 微波 信号 复用 电路 ， 该 复用 电路 由 五个 端口 构成 ， 五个 端口 分别 是 复合 端口 、 直流 端口 、 遥测 信号 端口 、 室外 单元 ( ODU ) 接收 中频 ( RX - IF ) 端口 和 室内 单元 ( IDU ) 发射 中频 ( TX - IF ) 端口 ， 其中 室外 单元 ( ODU ) 接收 中频 ( RX - IF ) 端口 和 室内 单元 ( IDU ) 发射 中频 ( TX - IF ) 端口 与 信号源 或 频谱仪 连接 ， 复合 端口 与 室内 单元 ( IDU ) 或 室外 单元 ( ODU ) 连接 。 本发明 具有 全面 分析 、 双模 应用 、 便携式 设计 、 广泛 适用性 、 成本 廉价 、 操作 简便 等 优势 。
