一种 水泥 基 混凝土 电加热 养护 系统 本 实用新型 涉及 水泥 基 混凝土 材料 的 养护 成型 领域 ， 特别 是 涉及 一种 水泥 基 混凝土 电加热 养护 系统 ， 包括 养护 室 ( 1 ) 、 加热 系统 、 加湿 系统 、 空气 循环系统 及 养护 室 控制箱 ； 其 特征 在于 ： 所述 养护 室 ( 1 ) 为 标准 养护 房 ， 是 提供 养护 操作 的 场所 ； 所述 加湿 系统 为 整个 养护 过程 提供 必须 要 的 水蒸气 ， 包括 加湿 水箱 ( 3 ) 及 控制 元器件 ； 所述 空气 循环系统 包括 循环 风机 ( 6 ) 、 风道 ( 7 ) 、 循环 风机 电机 ( 4 ) 、 风机 软 连接 等 ； 所述 空气 循环系统 是 将 加热 箱 ( 2 ) 产生 的 热量 及 加湿 水箱 ( 3 ) 产生 的 水蒸气 经过 循环 风机 ( 6 ) 的 带动 ， 经过 风道 ( 7 ) 再 将 房间内 空气 循环 回 加热 箱 ( 2 ) 。 所述 养护 室 控制箱 是 控制 加热 系统 、 加湿 系统 及 空气 循环系统 的 具体 执行 设备 。
