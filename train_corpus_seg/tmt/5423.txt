汽车 远程 数据 采集 系统 本 实用新型 公开 了 一种 汽车 远程 数据 采集 系统 ， 设置 有 中央处理器 ( 3 ) ， 该 中央处理器 ( 3 ) 经 无线通信 模块 ( 1 ) 连接 有 天线 ( 9 ) ， 其 关键在于 ： 所述 中央处理器 ( 3 ) 的 地理位置 输入 端 连接 有 GPS 模块 ( 5 ) ， 该 中央处理器 ( 3 ) 唤醒 端 连接 到 启动 ACC ( 7 ) 上 ， 该 中央处理器 ( 3 ) 的 数据 采集 端组 连接 有 CAN 通讯 模块 ( 2 ) 。 本 实用新型 能 远程 采集 公交车 时时 运行 数据 ， 车辆 状况 数据 ， 方便管理 ， 为 车辆 维修保养 提供 依据 和 参考 ， 能 提前 暴露 公交车 故障 情况 ， 降低 公交车 运行 风险 。
