具有 石墨 烯振膜 的 传声器 一种 具有 石墨 烯振膜 的 电容式 传声器 ， 包括 石墨 烯振膜 ( M ) 、 具有 通孔 的 背 极板 ( 4 ) ， 振膜 ( M ) 与 背 极板 ( 4 ) 之间 由 绝缘 环 ( 3 ) 相隔 ， 振膜 ( M ) 与 背 极板 ( 4 ) 之间 具有 间隔 d ， 振膜 ( M ) 与 背 极板 ( 4 ) 分别 通过 导电 组件 ( 5 ) 连接 至 供电 及 放大 组件 ( 6 ) ， 从而 振膜 ( M ) 与 背 极板 ( 4 ) 构成 电容器 ， 针对 形成 电容 结构 的 振膜 与 背 极板 的 不同 间距 d ， 采用 了 不同 的 加电 手段 以 实现 传声器 功能 ， 解决 了 如何 使 具有 石墨 烯振膜 的 传声器 正常 工作 的 技术 问题 。 本发明 的 传声器 的 灵敏度 、 频率响应 、 瞬态 响应 等 电声 特性 相比 传统 电容式 传声器 均 得到 较大 提升 。
