一种 一体化 绞肉机 本 实用新型 公开 了 一种 一体化 绞肉机 ， 包括 入料 槽 ( 1 ) 、 绞肉机 装置 ( 5 ) ， 还 包括 清洗 装置 ( 2 ) 、 切割 装置 ( 3 ) 、 输送 槽 ( 4 ) 、 过滤 槽 ( 6 ) ， 其中 ， 入料 槽 ( 1 ) 与 切割 装置 ( 3 ) 入口 相连接 ， 清洗 装置 ( 2 ) 安装 在 入料 槽 ( 1 ) 和 切割 装置 ( 3 ) 的 上端 ， 切割 装置 ( 3 ) 与 绞肉机 ( 5 ) 之间 通过 输送 槽 ( 4 ) 相连接 ， 过滤 槽 ( 6 ) 设置 在 输送 槽 ( 4 ) 的 底端 。 本 实用新型 结构 简单 操作 方便 ， 系统 的 流水线 ， 保证 绞肉 的 干净 卫生 和 细度 。
