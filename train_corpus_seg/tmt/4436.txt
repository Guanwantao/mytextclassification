一种 车载 天线 本 实用新型 公开 了 一种 车载 天线 ， 它 包括 天线 杆 ( 1 ) 、 支脚 ( 2 ) 和 底座 ( 3 ) ， 它 还 包括 太阳能 电池板 ( 8 ) 和 透光 板 ( 7 ) ， 所述 的 天线 杆 通过 螺母 ( 4 ) 和 螺杆 ( 5 ) 安装 在 支脚 ( 2 ) 上 ， 所述 的 支脚 ( 2 ) 焊接 在 底座 ( 3 ) 上 ， 所述 的 太阳能 电池板 ( 8 ) 安装 在 底座 ( 3 ) 上 ， 透光 板 ( 7 ) 覆盖 安装 在 太阳能 电池板 ( 8 ) 上 ， 通过 螺钉 ( 6 ) 固定 在 底座 上 。 该 车载 天线 集成 了 太阳能 电池板 能够 对 有源 天线 进行 供电 ， 方便 了 用户 外出 ， 同时 太阳能 电池板 更换 方便 ， 使用 太阳能 ， 清洁 环保 。
