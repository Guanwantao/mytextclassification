VPN 中 安全 代理 的 Web 服务 映射 方法 及其 系统 本发明 涉及 通信 领域 ， 公开 了 一种 VPN 中 安全 代理 的 Web 服务 映射 方法 及其 系统 。 简化 了 Web 响应 解析 和 改写 过程 ， 其 实现 简单 且 稳定性 和 效率 较 高 。 本发明 中 ， VPN 中 的 安全 代理 预先 设置 有 提供 Web 服务 与 安全 代理 的 域名 或 端口 的 映射 关系 ， 其中 不同 的 Web 服务 对应 不同 的 域名 或 端口 ； 当 安全 代理 收到 来自 客户端 的 Web 服务 请求 时 ， 根据 该 请求 中 的 域名 或者 接收 到 请求 的 端口 从 映射 关系 中 找到 对应 的 Web 服务 ， 并 将 该 请求 转发给 提供 该 Web 服务 的 Web 服务器 ； 当 安全 代理 收到 来自 Web 服务器 的 Web 响应 时 ， 根据 映射 关系 ， 将 Web 响应 中 指向 该 Web 服务 的 链接 替换成 指向 该 安全 代理 的 对应 域名 或者 端口 的 链接 ， 再 将 经 替换 的 Web 响应 转发给 客户端 。
