一种 养猪场 用 新型 自动 供料 装置 为了 解决 现有 养猪场 用 供料 系统 较为 复杂 的 问题 ， 本 实用新型 提出 一种 养猪场 用 新型 自动 供料 装置 ， 包括 储料 箱 ( 1 ) 、 输料 管 ( 2 ) 、 分料 刷 ( 3 ) 、 连接 装置 ( 4 ) 、 动力装置 ( 5 ) 、 分料 管 ( 6 ) 、 余料 收集 管 ( 7 ) 、 抽风机 ( 8 ) 、 鬃毛 刷 ( 9 ) 、 电源 ( 10 ) 、 导轨 ( 11 ) ； 在 所述 输料 管 ( 2 ) 的 内壁 ， 偏离 输料 管 ( 2 ) 与 储料 箱 ( 1 ) 、 余料 收集 管 ( 7 ) 及 各个 分料 管 ( 6 ) 连通 口 的 位置 设置 有 导轨 ( 11 ) ； 在 导轨 ( 11 ) 上 安装 有 可 沿着 其 运动 的 动力装置 ( 5 ) ； 动力装置 ( 5 ) 通过 连接 装置 ( 4 ) 连接 有 分料 刷 ( 3 ) ； 所述 分料 刷 ( 3 ) 为 圆周 带有 鬃毛 刷 ( 9 ) 的 圆形 硬质 薄板 ； 在 余料 收集 管 ( 7 ) 上 设置 有 抽风机 ( 8 ) 。
