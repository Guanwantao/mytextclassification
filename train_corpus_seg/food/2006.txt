紫外 分析仪 的 暗箱 本 实用新型 提供 了 一种 紫外 分析仪 的 暗箱 ， 包括 ： 一个 箱体 ( 10 ) 、 一个 载物台 ( 20 ) 、 一个 观察窗 ( 30 ) 和 一个 载物 架 ( 40 ) 。 载物台 ( 20 ) 固定 于 箱体 ( 10 ) 的 内部 ， 观察窗 ( 30 ) 设置 于 箱体 ( 10 ) 的 侧壁 。 载物 架 ( 40 ) 包括 一个 用于 盛放 样品 的 架体 ( 42 ) 。 箱体 ( 10 ) 的 侧壁 处开 有 一个 与 架体 ( 42 ) 形成 间隙 配合 的 插孔 ( 12 ) ， 架体 ( 42 ) 可 通过 插孔 ( 12 ) 插入 或拉出 箱体 ( 10 ) 并 在 载物台 ( 20 ) 上 移动 。 本 实用新型 的 紫外 分析仪 的 暗箱 ， 可 通过 架体 与 插孔 的 配合 ， 有效 阻挡 箱体 内部 的 紫外线 外泄 ， 如此 便 无需 在 更换 样品 时 关闭 紫外 灯 ， 即可 延长 紫外 灯管 的 寿命 。
