格 层状 冰淇淋 的 挤出 装置 格 层状 冰淇淋 的 挤出 装置 ， 它 包括 ： 第一 储料 罐 ( 10 ) 、 第二 储料 罐 ( 20 ) 、 输料 轴 ( 30 ) 、 成型 器 ( 40 ) 、 成型 罩 ( 50 ) 和 可 驱动 输料 轴 ( 30 ) 旋转 的 驱动 组件 ( 60 ) ； 第一 储料 罐 ( 10 ) 具有 第一 料腔 ( 12 ) ； 第二 储料 罐 ( 20 ) 具有 第二 料腔 ( 22 ) ； 输料 轴 ( 30 ) 穿 设于 第一 储料 罐 和 第二 储料 罐 ， 且 具有 输料 腔 ( 32 ) 、 出料 口 和 一个 连通 输料 腔 ( 32 ) 和 第一 料腔 ( 12 ) 的 输料 口 ( 34 ) ； 输料 轴 ( 30 ) 可 驱动 连接 于 成型 器 ( 40 ) ， 成型 器 ( 40 ) 具有 一个 出料 端 和 一个 可 连接 输料 轴 ( 30 ) 的 进料 端 ； 成型 罩 ( 50 ) 设置 于 成型 器 ( 40 ) 的 出料 端 一侧 。 该 挤出 装置 可以 挤出 两种 不同 的 冰淇淋 料 并 形成 上下 错层 的 格 层状 冰淇淋 。
