软 提手 的 安装 装置 本 实用新型 提供 了 一种 软 提手 的 安装 装置 ， 其 包括 提手 模具 ( 10 ) 、 挡片 模具 ( 20 ) 和 安装 件 ( 30 ) 。 提手 模具 包括 两个 滑槽 ( 12 ) ， 两个 滑槽 相向 且 相互 对称 设置 ， 各 滑槽 具有 一个 入口 端 ( 14 ) 和 一个 出口 端 ( 16 ) ， 两个 滑槽 的 入口 端的 距离 大于 两个 滑槽 的 出口 端的 距离 。 挡片 模具 朝向 提手 模具 出口 端的 一侧 具有 一个 定位 部 ( 22 ) ， 定位 部上 设有 两个 组装 空间 ( 24 ) 。 安装 件 包括 两个 弹性 插条 ( 32 ) ， 两个 弹性 插条 可 分别 推动 提手 ( 40 ) 的 两个 连接 部 ( 42 ) ， 使 提手 沿 滑槽 从 滑槽 的 入口 端 滑向 出口 端 ， 并 可 使 提手 的 两个 连接 部分 别 穿过 挡片 ( 60 ) 的 固定 孔 ( 62 ) 进入 组装 空间 内 。 软 提手 的 安装 装置 可以 实现 软 提手 的 机械 安装 手段 。
