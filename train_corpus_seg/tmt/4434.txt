一种 支持 远程 监控 OBD 终端装置 本 实用新型 公开 了 一种 支持 远程 监控 OBD 终端装置 ， 它 包括 微处理器 ( 3 ) 、 GPS 定位器 ( 4 ) 采集 接口 ( 1 ) 、 OBD 检测 芯片 ( 2 ) 、 通讯 模块 ( 9 ) 和 备用 电池 ( 10 ) ， OBD 检测 芯片 ( 2 ) 通过 采集 接口 ( 1 ) 采集 汽车 运行 数据 ， 通过 串口 输入 到 微处理器 ( 3 ) ， 微处理器 ( 3 ) 通过 串口 分别 连接 FLASH 装置 ( 5 ) 、 指示灯 ( 7 ) 和 通讯 模块 ( 9 ) ， 3D 加速度 传感器 ( 6 ) 通过 SPI 接口 连接 微处理器 ( 3 ) ， Debug 调试 装置 ( 8 ) 、 GPS 定位器 ( 4 ) 分别 与 微处理器 ( 3 ) 连接 。 本 实用新型 的 有益 效果 是 ： 提供 车辆 行驶 中 的 各项 参数 ， 设置 车辆 安全 区域 ， 当 车辆 移出 设置 区域 立即 报警 ， 支持 远程 监控 ， 通过 GPS 定位器 和 通讯 网络 对 车辆 进行 实时 监控 。
