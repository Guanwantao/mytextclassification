一种 基于 物 联网 Android 平台 的 空气 净化器 无线 监测 系统 本 实用新型 公开 了 一种 基于 物 联网 Android 平台 的 空气 净化器 无线 监测 系统 ， 包括 ： 空气质量 检测 模块 ( 1 ) 、 ZigBee 无线 发送 模块 ( 2 ) 、 ZigBee 无线通讯 网络 ( 3 ) 、 INTERNET 网络 ( 4 ) 、 监控 中心 ( 5 ) 、 手机 客户端 ( 6 ) 、 ZigBee 无线 接收 模块 ( 7 ) 、 空气质量 控制 模块 ( 8 ) ； 空气质量 检测 模块 ( 1 ) 将 室内 温度 、 湿度 和 气体 的 信息 进行 采集 ， 由 ZigBee 无线 发送 模块 ( 2 ) 进行 无线 发送 ， 经 ZigBee 无线通讯 网络 ( 3 ) 和 INTERNET 网络 ( 4 ) 发送至 监控 中心 ( 5 ) ， 再 经 嵌入式 Wi - Fi 物 联网 发送至 手机 客户端 ( 6 ) ， 手机 客户端 ( 6 ) 将 传送 过来 的 信息 经过 分析 处理 后 ， 将 是否 要 对 空气质量 进行 控制 的 信息 经 嵌入式 Wi - Fi 物 联网 传送 至 监控 中心 ， 再经 INTERNET 网络 ( 4 ) 和 ZigBee 无线通讯 网络 ( 3 ) 传送 至 ZigBee 无线 接收 模块 ( 7 ) 和 空气质量 控制 模块 ( 8 ) ， 控制 空气 净化器 是否 要 对 空气质量 进行 控制 ， 实现 全 自动化 和 智能化 的 室内 空气净化 。 本 实用新型 结构 简单 、 性能 稳定 、 操作 使用方便 。
