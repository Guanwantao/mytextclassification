型钢 混凝土 梁 与 梁上 混凝土 柱 的 连接 节点 本 实用新型 公开 了 一种 型钢 混凝土 梁 与 梁上 混凝土 柱 的 连接 节点 ， 型钢 混凝土 梁 ( 7 ) 中 设置 有型 钢梁 ( 6 ) ， 梁 上 混凝土 柱 ( 8 ) 中 设置 有 若干 根竖 向 钢筋 ( 1 ) ， 其 特征 在于 ： 所述 型 钢梁 ( 6 ) 的 翼缘板 ( 5 ) 上 固定 有 若干个 螺栓 套筒 ( 2 ) ， 所述 螺栓 套筒 ( 2 ) 的 轴向 垂直于 翼缘板 ( 5 ) ， 所述 翼缘板 ( 5 ) 正上方 的 竖向 钢筋 ( 1 ) 插入 固定 在 螺栓 套筒 ( 2 ) 中 ， 翼缘板 ( 5 ) 两侧 的 竖向 钢筋 ( 1 ) 锚入 型 钢梁 ( 7 ) 侧部 的 混凝土 中 。 由于 竖向 钢筋 通过 螺栓 套筒 与 型 钢梁 连接 ， 有效 避免 了 型 钢梁 的 翼缘板 上 开洞 或 直接 在 型 钢梁 上 焊接 的 情况 。 本 实用新型 设计 巧妙 ， 不 影响 型 钢梁 的 强度 ， 性能 优良 ， 方便 实用 。
