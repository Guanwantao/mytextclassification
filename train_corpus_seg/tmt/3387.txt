一种 开环 控制 的 遥控 车位锁 本发明 提供 了 一种 开环 控制 的 遥控 车位锁 ， 包括 安装 座 ( 1 ) ； 在 所述 安装 座 ( 1 ) 上 设置 有 控制 装置 ( 10 ) 、 电机 ( 2 ) 、 丝杠 ( 3 ) 和 底杆 ( 6 ) ， 其中 所述 控制 装置 ( 10 ) 上 配置 有 可拆卸 的 蓄电池 ( 8 ) ， 在 电机 靠近 丝杠 ( 3 ) 的 端面 处 安装 有 位置 读取器 ( r ) ， 在 所述 丝杠 ( 3 ) 上 设置 有 与 位置 读取器 ( r ) 相对 应 的 位置 信号 产生 装置 ( c ) ， 所述 位置 读取器 ( r ) 通过 读取 所述 位置 信号 产生 装置 ( c ) 能够 得到 丝杠 的 转动 圈数 ； 无线 信号 接收 模块 ( 15 ) 和 车辆 离开 传感器 模块 ( 16 ) 分别 安装 在 所述 可动 面板 ( 5 ) 的 上端 两侧 以便 于 接收 信号 。
