滑动式 防撞 护栏 滑动式 防撞 护栏 是 一种 能够 有效 提高 防撞 性能 和 减少 人员 车辆 撞击 损伤 的 新型 交通 防护 设施 。 它 由 立柱 ( 1 ) 、 槽钢 ( 2 ) ( 3 ) 、 转子 轴 钢棒 ( 4 ) 、 滑动 转子 ( 5 ) 、 紧固件 ( 6 ) 、 上盖 ( 7 ) 、 螺栓 ( 8 ) 等 部分 构成 ， 安装 时 首先 埋入 立柱 ( 1 ) ， 之后 安装 下 槽钢 ( 3 ) 和 上 槽钢 ( 2 ) ， 转子 轴 钢棒 ( 4 ) 由下 槽钢 ( 3 ) 向上 穿 入 ， 套入 滑动 转子 ( 5 ) 后 ， 再 由 上 槽钢 ( 2 ) 穿 出 ， 用 扳手 拧紧 转子 轴 上下 锁紧 螺母 ( 6 ) ， 最后 安装 上盖 ( 7 ) 并 拧紧 固定 螺栓 ( 8 ) 。 滑动 转子 ( 5 ) 表面 可 着色 成 交通 引导 标志 和 警示 标志 。 滑动式 防撞 护栏 基于 “ 宽容 型 ” 设计 理念 ， 利用 了 防撞 与 滑动 导向 相结合 的 原理 ， 主要 应用 于 “ 弯急 、 坡 陡 、 路窄 、 沟深 ” 的 路侧 高危 路段 。
