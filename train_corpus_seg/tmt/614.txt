基于 业务流程 可视化 的 SQL 性能 监控 方法 本发明 公开 了 一种 基于 业务流程 可视化 的 SQL 性能 监控 方法 ， 包括 如下 步骤 ： a ) 将 业务流程 按 数据流 向 划分 为 多个 业务 节点 ， 并 进行 可视化 图形 显示 ； b ) 在 每个 业务 节点 配置 与 该 业务 节点 相关联 的 SQL 语句 以及 需要 访问 的 数据表 ， 实现 业务流程 和 数据库 底层 SQL 的 透明化 关联 ； c ) 定期 向 数据库 收集 影响 业务 性能 的 SQL 语句 ， 并 抽取 SQL 语句 中 执行 的 相关 数据表 名称 ； d ) 查找 与 抽取 到 的 相关 数据表 名称 相关联 的 业务 节点 ， 实现 业务 故障 的 定位 。 本发明 实现 了 业务流程 和 数据库 底层 SQL 的 透明化 关联 ， 并 将 业务流程 进行 可视化 图形 显示 ， 从而 提供 了 有效 的 业务 问题 预警 , 故障 定位 精准度 变高 ， 运维 成本 大幅度 下降 。
