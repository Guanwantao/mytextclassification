模块化 组装 房屋 体系 本 实用新型 公布 了 一种 模块化 组装 房屋 体系 ， 其 特征 在于 ： 它 包括 一个 或 多个 模块化 制作 而成 的 房屋 ( 1 ) ， 且 两层 以内 的 所述 的 房屋 ( 1 ) 在 横向 无限 延伸 ， 每个 所述 的 房屋 ( 1 ) 由 钢结构 框架 ( 2 ) 、 冷弯 薄壁 型钢 体系 墙板 ( 3 ) 、 冷弯 薄壁 型钢 体系 楼板 ( 4 ) 和 冷 弯 薄壁 型钢 体系 屋面 ( 5 ) 组成 ； 所述 的 钢结构 框架 ( 2 ) 包括 上下 互相 平行 的 横梁 ( 2.1 ) 和 垂直 连接 的 立柱 ( 2.2 ) ， 所述 的 横梁 ( 2.1 ) 和 立柱 ( 2.2 ) 围成 的 八个 边角 均 用 连接件 ( 7 ) 连接 ； 它 克服 了 现有 技术 中折装 不 方便 ， 组合 性 不好 ， 不 适合 复杂 的 环境 的 缺点 ， 具有 可拆卸 、 可 互换 、 可 增减 的 优点 。
