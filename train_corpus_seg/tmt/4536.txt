一种 方便 装配 的 控制器 本 实用新型 的 目的 在于 提供 一种 的 方便 拆卸 又 美观 的 控制器 扣合 结构 ， 不但 满足 了 控制器 免 螺丝 安装 的 技术 方案 ， 而且 对 该 产品 内部结构 的 改进 又 不 影响 该 产品 的 整体 外观设计 ， 具有 结构 简单 、 实用性 高等 特点 。 其 主要 内容 为 一种 方便 装配 的 控制器 ， 主要 包括 控制器 第一 壳体 ( 1 ) ， 控制器 第二 壳体 ( 2 ) ， 控制 主板 ( 3 ) 组成 ， 其 特征 在于 ： 所述 控制器 第一 壳体 ( 1 ) 设置 有 扣位 ( 11 ) 、 第一 空腔 ( 12 ) 、 主板 定位 柱 ( 13 ) 、 主板 卡扣 ( 14 ) 等 组成 ； 所述 控制器 第二 壳体 ( 2 ) 设置 有 弹片 ( 21 ) 和 第二 空腔 ( 22 ) 等 组成 ； 所述 第一 壳体 与 第二 壳体 上 的 弹片 及 空腔 位置 及 数量 对应 设置 。
