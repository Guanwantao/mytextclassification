基于 phthon 构建 桌面版 C / S 自动化 运维 平台 的 方法 本发明 公开 了 一种 基于 phthon 构建 桌面版 C / S 自动化 运维 平台 的 方法 ， 包括 ： 设置 一台 主 服务器 ， 并 在 主 服务器 上 部署 WEB 页面 、 python 环境 和 数据库 ； 编写 第一 python 脚本 获取 被控 服务器 的 硬件 信息 和 网络 配置 信息 ； 对 被控 服务器进行 监测 ， 根据 对应 的 服务 编写 用于 监测 的 第二 python 脚本 ； 根据 被控 服务器 的 日志 格式 ， 编写 用于 日志 统计 与 分析 的 第三 python 脚本 ； 在 主 服务器 上 通过 python 程序 连接 数据库 查询 数据 ， 并 将 数据 呈现 在 WEB 页面 上 。 本发明 能够 方便 地 通过 WEB 页面 直接 查到 每台 被控 服务器 日志 分析 的 数据 以及 每台 被控 服务器 上 运行 的 服务 状态 ， 提高 了 运维 工作效率 ， 避免 了 运维 工作 重复性 。
