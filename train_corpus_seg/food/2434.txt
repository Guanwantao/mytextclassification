颗粒 检测 筛及 颗粒 检测 装置 颗粒 检测 筛 ( 10 ) ， 其 包括 第一 筛体 ( 20 ) 和 第二 筛体 ( 30 ) 。 第一 筛体 为 圆柱形 ， 其 包括 第一 底部 筛网 ( 25 ) 和 第一 侧壁 筛网 ( 27 ) 。 第一 侧壁 筛网 连接 于 第一 底部 筛网 ， 且 与 第一 底部 筛网 围成 第一 容腔 ( 21 ) 。 第一 侧壁 筛网 未 连接 第一 底部 筛网 的 一端 形成 第一 入料 口 ( 23 ) 。 第二 筛体 为 圆锥台 形 ， 其 顶部 直径 大于 底部 直径 。 第二 筛体 包括 第二 底部 筛网 ( 35 ) 和 第二 侧壁 筛网 ( 37 ) 。 第二 侧壁 筛网 连接 于 第二 底部 筛网 ， 且 与 第二 底部 筛网 围成 第二 容腔 ( 31 ) 。 第二 侧壁 筛网 未 连接 第二 底部 筛网 的 一端 形成 第二 入料 口 ( 33 ) 。 第二 筛体 的 底部 能够 连接 第一 筛体 的 顶部 以 封闭 第一 入料 口 。 该 颗粒 检测 筛 操作 简便 。 本 实用新型 还 提供 了 颗粒 检测 装置 。
