封装 机构 及 塑杯 封合 系统 封装 机构 包括 支架 ( 10 ) 、 驱动 气缸 ( 20 ) 、 封装 头 ( 30 ) 和 限位 杆 ( 80 ) 。 支架 设有 限位 孔 ( 11 ) 。 驱动 气缸 设 于 支架 且 具有 驱动 杆 ( 21 ) 。 封装 头 包括 连接件 ( 40 ) 、 止 退板 ( 50 ) 、 导向 组件 ( 60 ) 和 热封 压头 ( 70 ) 。 连接件 包括 第一 、 第二 和 第三 连接 部 。 止 退板 连接 于 第三 连接 部且 具有 导向 通孔 ( 51 ) 。 导向 组件 包括 导向 杆 ( 61 ) 、 止 退件 ( 63 ) 和 压缩 弹簧 ( 65 ) 。 导向 杆 穿 设于 止 退板 。 止 退件 连接 导向 杆 。 压缩 弹簧 套设 导向 杆 。 热封 压头 连接 导向 杆 并 与 止 退板 挤压 压缩 弹簧 。 限位 杆 藉由 限位 孔穿 设 支架 并 连接 止 退板 ， 限位 孔 的 孔壁 与 限位 杆 之间 具有 间隙 。 该 封装 机构 保证 了 较 高 的 封合率 和 易 撕性 。 此外 还 提供 了 塑杯 封合 系统 。
