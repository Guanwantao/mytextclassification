线性 恒流 调制 电路 本发明 涉及 一种 线性 恒流 调制 电路 。 该 线性 恒流 调制 电路 包括 电压 检测 电路 、 功率管 ( M1 ) 、 开关 管 ( M2 ) 和 若干 电阻 ( Rext ) ； 所述 功率管 ( M1 ) 与 所述 开关 管 ( M2 ) 串联 连接 ； 所述 开关 管 ( M2 ) 与 所述 若干 电阻 ( Rext ) 并联 连接 ； 所述 电压 检测 电路 检测 所述 功率管 ( M1 ) 两端 的 电压 差 ( VM1 ) ， 所述 电压 差 ( VM1 ) 超过 预设 的 阈值电压 ( VMT ) 时 ， 所述 电压 检测 电路 控制 所述 开关 管 ( M2 ) 为 截止 状态 。 本发明 消除 了 功率管 工作 功率 过高时 对 恒流 电路 带来 的 不利 影响 ， 提高 线性 恒流 调制 电路 工作 的 稳定性 与 可靠性 。
