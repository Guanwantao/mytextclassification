一种 便携式 车载 GPS 导航仪 本 实用新型 公开 了 一种 便携式 车载 GPS 导航仪 ， 它 包括 外壳 ( 1 ) 、 触摸 显示屏 ( 2 ) 、 多个 控制 按钮 ( 3 ) 、 防滑条 ( 4 ) 、 开 关键 ( 7 ) 和 存储卡 槽 ( 8 ) ， 所述 的 触摸 显示屏 ( 2 ) 和 控制 按钮 ( 3 ) 设置 在 外壳 ( 1 ) 的 正面 ， 控制 按钮 ( 3 ) 设置 在 触摸 显示屏 ( 2 ) 的 一侧 ， 防滑条 ( 4 ) 设置 在 外壳 ( 1 ) 的 两侧 ， 外壳 ( 1 ) 的 侧面 设置 有开 关键 ( 7 ) 和 存储卡 槽 ( 8 ) 。 本 实用新型 具有 触摸 操作 、 夜间 照明 、 体积小 、 屏幕 大 等 特点 ， 可 连接 耳机 更加 节能 省 电 ， 支持 插入 存储卡 增加 导航仪 存储容量 ， 具有 防滑条 克服 手持 时 容易 滑落 的 困难 ， 具有 挂绳 接口 ， 增加 导航仪 的 便携性 ， 具有 标准 的 USB 充电 接口 ， 便于 导航仪 在 车内 及 户外 充电 。
