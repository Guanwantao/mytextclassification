人体 久坐 状态 检测 装置 中 的 LED 灯带 组件 的 安装 结构 本 实用新型 公开 了 一种 人体 久坐 状态 检测 装置 中 的 LED 灯带 组件 的 安装 结构 ， 包括 LED 灯带 组件 、 由 上 外壳 ( 1 ) 和 下 外壳 ( 2 ) 连接 而成 的 外 壳体 及 由 上 内壳 ( 3 ) 和 下 内壳 ( 4 ) 连接 而成 的 内 壳体 ， LED 灯带 组件 包括 带有 若干 LED 灯 的 环形 灯带 ( 5 ) 及 用于 分散 LED 灯光 线 的 扩散 环 ( 6 ) ， 环形 灯带 ( 5 ) 套接 在 上 内壳 ( 3 ) 的 外部 且 通过 上 外壳 ( 1 ) 压紧 在 上 内壳 ( 3 ) 上 ， 扩散 环 ( 6 ) 设置 在 上 外壳 ( 1 ) 和 下 外壳 ( 2 ) 之间 且 与 上 内壳 ( 3 ) 和 下 内壳 ( 4 ) 固定 ， 环形 灯带 ( 5 ) 与 扩散 环 ( 6 ) 之间 设有 间隙 ； 本 实用新型 能够 使 LED 灯带 发出 的 光 变得 均匀 ， 且 具有 LED 灯带 装配 方便 的 优点 。
