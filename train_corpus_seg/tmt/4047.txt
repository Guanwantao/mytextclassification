基于 特征 模式识别 的 sql 注入 检测 方法 本发明 公开 了 一种 基于 特征 模式识别 的 sql 注入 检测 方法 ， 包括 如下 步骤 ： ( 1 ? 1 ) 基于 sql 语法 解析 和 sql 语义 解析 生成 sql 注入 特征 模式 抽象 语法 树库 ； ( 1 ? 2 ) 获取 应用程序 到 数据库 的 网络流量 ， 根据 数据库 协议 解析 网络流量 ， 从中 提取 完整 的 sql 语句 ； ( 1 ? 3 ) 对 获取 的 sql 语句 进行 SQL 语句 语法 解析 ， 获取 sql 语句 的 语法 单元 ； ( 1 ? 4 ) 对 sql 语句 的 语法 单元 进行 sql 语义 解析 ， 执行 语义 计算 ， 生成 计算 简化 过 的 目标 抽象 语法 树 ； 本发明 具有 减少 了 误报 、 漏报 ， 提高 了 工作效率 的 特点 。
