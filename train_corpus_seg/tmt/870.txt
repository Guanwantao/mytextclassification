智能 交通 一体机 本 实用新型 公开 了 一种 智能 交通 一体机 ， 包括 箱体 ( 1 ) ， 箱体 ( 1 ) 内部 设有 CPU 主板 ， 箱体 ( 1 ) 上下 两端 面开 有 凹槽 ( 2 ) ， 所述 箱体 前端 面内 还 设置 有 状态 指示灯 ( 3 ) 、 GPRS 天线 接口 ( 4 ) 、 摄像头 接口 ( 5 ) 、 操作 手柄 接口 ( 6 ) 、 扩展 接口 ( 7 ) 和 电源接口 ( 9 ) ， 所述 操作 手柄 接口 ( 6 ) 、 扩展 接口 ( 7 ) 和 电源接口 ( 9 ) 通过 螺钉 ( 8 ) 固定 在 箱体 ( 1 ) 前端 面 。 本 实用新型 的 优点 在于 ： 本 实用新型 设计 的 这种 智能 交通 一体机 ， 不仅 集 “ 报 站 、 调度 管理 、 车载 监控 ” 等 功能 为 一体 ， 且 体积小 、 成本低 。
