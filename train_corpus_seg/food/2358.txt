果粒 自动 清洗 装置 果粒 自动 清洗 装置 ， 包括 一个 水槽 ( 10 ) 、 一个 弹性 支撑 机构 ( 20 ) 、 一个 筛体 ( 30 ) 、 和 两个 振动 件 ( 40 ) 。 水槽 有 一个 排水口 ( 12 ) 。 弹性 支撑 机构 设置 于 水槽 内 ， 其 包括 数个 弹性 件 ( 22 ) 和 一个 支撑 盘 ( 24 ) ， 支撑 盘 包括 一个 安装 顶面 ( 25 ) 和 一个 安装 底面 ( 27 ) ， 安装 底面 上 有 数个 锁紧 件 ( 28 ) ， 弹性 件 沿周 向 均匀 布于 支撑 盘 的 安装 底面 。 筛体 由 锁紧 件 固定 于 安装 底面 。 振动 件 分别 包括 一个 电机 ( 42 ) 、 一个 传动轴 ( 44 ) 和 一个 偏心轮 ( 46 ) 。 传动轴 与 电机 连接 ， 偏心轮 与 传动轴 连接 ， 振动 件 分别 设 于 安装 顶 面上 ， 各 传动轴 的 轴向 互相 垂直 。 上述 果粒 自动 清洗 装置 具有 多种 振动 方式 ， 能够 更 有效期 地 清洗 果粒 上 的 糖分 。
