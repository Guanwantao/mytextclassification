冰淇淋 的 切割 装置 冰淇淋 的 切割 装置 包括 一个 凸轮 ( 10 ) 、 一个 横梁 ( 12 ) 、 一个 滑块 ( 14 ) 、 一个 驱动 杆 ( 16 ) 、 和 一个 切割 拉杆 ( 18 ) 。 凸轮 的 型 线 包括 切割 段 ( 102 ) 和 回程 段 ( 104 ) ， 切割 段 上 各点 到 凸轮 的 转动 中心 的 距离 相等 ， 回程 段 上 各点 到 凸轮 的 转动 中心 的 最大 距离 小于 切割 段 上 各点 到 凸轮 的 转动 中心 的 距离 。 横梁 具有 一个 固定 端 ( 122 ) 、 一个 摆动 端 ( 124 ) 、 和 一个 导向 槽 ( 126 ) 。 横梁 可绕 固定 端 运动 。 凸轮 可 驱动 摆动 端 使 其绕 固定 端 运动 。 滑块 能 在 导向 槽 中 滑动 。 驱动 杆 的 两端 分别 枢接 于 凸轮 和 滑块 。 切割 拉杆 具有 一个 固接 于 滑块 的 连接端 ( 184 ) 和 一个 能 切割 冰淇淋 的 切割 端 ( 182 ) 。 该 切割 装置 结构 简单 ， 运行 的 稳定性 较 好 。
