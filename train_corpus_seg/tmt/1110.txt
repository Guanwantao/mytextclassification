一种 三 通道 全息 声场 录音 设备 本 实用新型 提出 一种 三 通道 全息 声场 录音 设备 ， 包括 外壳 ( 11 ) 、 仿真 耳廓 ( 12 ) ， 外壳 ( 11 ) 内 封装 有 两只 耳道 传声器 ( 101 ) ， 设置 于 仿真 耳廓 ( 12 ) 的 耳道 内 ， 特征 在于 ： 在 外壳 ( 11 ) 上 的 中间 位置 与 耳廓 正对 方向 相一致 的 方向 上 还 设置 有 中置 传声器 ( 105 ) ， 由 模数转换器 ADC ( 102 ) 用于 将 传声器 ( 101 ) 拾取 的 双耳 模拟信号 及 中置 传声器 ( 105 ) 拾取 的 中间 模拟信号 转换 为 双耳 数字信号 L 、 R 及 中间 数字信号 C ， 还 包括 全息 声场 信号处理 收发 单元 ( 103 ) ， 接收 上述 双耳 数字信号 L 、 R 及 中间 数字信号 C ， 对 该 双耳 数字信号 L 、 R 及 中间 数字信号 C 进行 处理 后 或者 不 经过 处理 直接 以 无线 方式 发送至 外部 的 移动 终端 ( 2 ) 进行 记录 。 使用 该 设备 有效 提升 了 全息 声场 信号 的 重放 效果 以及 其 记录 、 分享 及 通讯 能力 。
